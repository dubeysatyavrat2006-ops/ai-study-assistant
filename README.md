# 📚 AI Study Assistant

A modern AI-powered web application that helps students learn efficiently by converting notes into summaries, quizzes, and flashcards.

---

## 🚀 Features

* 📄 Upload PDF or paste notes
* ✨ Generate concise summaries
* ❓ Create quiz questions automatically
* 🧠 Generate flashcards for revision
* ⚡ Fast AI responses
* 🎨 Modern UI with animations

---

## 🛠 Tech Stack

**Frontend**

* HTML, CSS, JavaScript
* PDF.js (for PDF parsing)

**Backend**

* Node.js
* Express.js

**AI Integration**

* OpenRouter API (LLaMA / OpenChat models)

---

## 📂 Project Structure

```
ai-study-assistant/
│
├── server.js        # Backend server
├── package.json     # Dependencies
├── index.html       # Frontend UI
├── .env             # API key (not uploaded)
```

---

## ⚙️ Setup Instructions

### 1. Clone repository

```
git clone https://github.com/your-username/ai-study-assistant.git
cd ai-study-assistant
```

### 2. Install dependencies

```
npm install
```

### 3. Add environment variables

Create `.env` file:

```
OPENROUTER_API_KEY=your_api_key_here
```

### 4. Run backend

```
node server.js
```

### 5. Run frontend

Open `index.html` in browser
